# koa2后台设计

## 安装

```shell
yarn install
```

## 启动

```shell
yarn run start
```

## 使用

```shell
curl http://localhost:30000/api/user/
```

![user api](./img/start.png)

## 测试

```shell
yarn run test
```

![test](./img/test.png)
